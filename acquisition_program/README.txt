Setting up the environment:  
Download Anaconda and create a python 3.8 environment and name it "beta_juvenile_acquisition_env"

Make an account with Spinnaker at:
https://www.teledynevisionsolutions.com/products/spinnaker-sdk/?model=Spinnaker%20SDK&vertical=machine%20vision&segment=iis
Download the Spinnaker 4.2 for python 3.8 and install the PySpin package in your beta_juvenile_acquisition_env

Open AnacondaPrompt and activate the beta_juvenile_acquisition_env
Navigate to the acquisition program and run the following command, "pip install -r requirements.txt"

set the exp_name on line 31 of run.py.
type "python run.py" in your terminal.
press enter.
